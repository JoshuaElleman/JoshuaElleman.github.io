# Enhancement One

## Artifact Overview
Artifact one is an app that calculates earned interest and total account value for a banks savings account. It was created in C++ for CS 210 during October of 2023. Several components of this artifact show my abilities in software development. The separation of the application into classes highlights an understanding of object-oriented programming. The Banking class encapsulates business logic such as calculating balances and interest earned while the BankingApp class handles user interface and input validation.

## Justification for Inclusion in ePortfolio
The artifact was improved in several ways for this enhancement. Originally the C++ version of the application relied on manual formatting and less structured input handling. In the enhancement I rewrote the program in Java and improved readability, modularity, and maintainability. I did this by leveraging Java’s built in libraries. I also refined the formatting of the outputs to ensure consistency and accuracy and implemented safer input handling to prevent runtime errors. 

## Outcome Alignment
With this enhancement I met several of the course outcomes. The course outcomes I met with this enhancement are Employ strategies for building collaborative environments that enable diverse audiences to support organizational decision-making in the field of computer science and demonstrate an ability to use well-founded and innovative techniques, skills, and tools in computing practices for the purpose of implementing computer solutions that deliver value and accomplish industry-specific goals which were outcomes I planned to meet.

## Reflection on the Enhancement Process
One of the most important things I learned in the process of enhancing this artifact was how to translate logic between programming languages while accounting for differences in syntax, structure, and best practices. I had to rethink how to handle input handling and output formatting in the conversion from C++ to Java. I also gained a deeper understanding of object-oriented programming and removed some methods that were not being used in the original code. The main challenge I faced was configuring the development environment, particularly dealing with Java module system errors in Eclipse. Resolving these issues required troubleshooting configurations and helped me gain an understanding how Java classes differ from C++ classes.
